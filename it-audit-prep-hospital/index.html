สาย IT โรงพยาบาล

จำ 5 เรื่องหลัก  ·  ตอบให้ถูกสูตร  ·  รอดทุกสถานการณ์

5 เรื่องที่ต้องจำให้ได้ก่อนเข้าหน้างาน
1ระบบหลัก (HIS)
2Backup ข้อมูล
3Cybersecurity / PDPA
4Network / VPN
5ส่งข้อมูล สธ.
1. ระบบสารสนเทศโรงพยาบาล (HIS)
HOSxP / JHCIS / ระบบหลักการให้บริการ
คำถามที่เจอบ่อย
  • ใช้ระบบอะไร (เช่น HOSxP / JHCIS / อื่นๆ)
  • ระบบล่มบ่อยไหม มีผลกับการให้บริการอย่างไร
  • มีการสำรองข้อมูล (Backup) อย่างไร
แนวตอบที่ดี
  • ใช้ระบบ … เป็นระบบหลักในการให้บริการผู้ป่วย
  • มีการสำรองข้อมูลทุกวัน และเก็บแยกเครื่อง / แยกสถานที่
  • มีแผนรองรับกรณีระบบล่ม เช่น ใช้งาน manual ชั่วคราว
ถ้าตอบไม่ได้: "ขออนุญาตตรวจสอบข้อมูลเชิงเทคนิคเพิ่มเติม และจะรายงานให้ทราบภายหลังครับ"
2. Cybersecurity / PDPA
ความปลอดภัยข้อมูล · คุ้มครองข้อมูลส่วนบุคคล
คำถามยอดฮิตมาก
  • มีนโยบายความปลอดภัยข้อมูลไหม
  • ป้องกันข้อมูลผู้ป่วยรั่วไหลอย่างไร
  • มีการอบรมบุคลากรเรื่อง PDPA ไหม
แนวตอบที่ดี
  • มีนโยบายด้าน Cybersecurity และ PDPA ของหน่วยงาน
  • มีการกำหนดสิทธิ์การเข้าถึงข้อมูล (User / Password / Role)
  • มี Firewall / Antivirus / Backup ป้องกันข้อมูล
  • มีการอบรมเจ้าหน้าที่เป็นระยะอย่างสม่ำเสมอ
ถ้าตอบไม่ได้: "ในส่วนรายละเอียดเชิงนโยบาย ขออนุญาตประสานผู้รับผิดชอบหลัก และรายงานเพิ่มเติมครับ"
3. ระบบ Network / Internet
Backup Link · VPN · การเชื่อมต่อหน่วยงาน
คำถามที่อาจโดน
  • ระบบอินเทอร์เน็ตมีปัญหาไหม
  • มีสำรอง (Backup Link) ไหม
  • เชื่อมต่อ รพ.สต. / หน่วยงานอื่นอย่างไร
แนวตอบที่ดี
  • มีการเชื่อมต่อผ่าน VPN / Network ของ สธ.
  • มีการควบคุมผ่าน Firewall
  • มีการตรวจสอบระบบ Network สม่ำเสมอ
เคล็ดลับ: ถ้ามีของจริง เช่น VPN / Fiber / 4G Backup → ตอบไปเลยจะดูดีมาก
4. การเชื่อมโยงข้อมูล (Digital Health)
HDC · 43 แฟ้ม · MOPH Connect
คำถามที่ชอบถาม
  • ส่งข้อมูล 43 แฟ้มไหม ครบถ้วนหรือเปล่า
  • เชื่อม HDC / MOPH Connect ได้ไหม
  • ข้อมูลทันสมัยหรือไม่
แนวตอบที่ดี
  • มีการส่งข้อมูลเข้าสู่ระบบ HDC อย่างสม่ำเสมอ
  • มีการตรวจสอบคุณภาพข้อมูล (Data Quality) อย่างต่อเนื่อง
  • มีผู้รับผิดชอบดูแลข้อมูลโดยตรง
5. อุปกรณ์ / Software
ลิขสิทธิ์ · ความเพียงพอ · แผนจัดหา
คำถามที่อาจโดน
  • ใช้ Software ถูกลิขสิทธิ์ไหม
  • เครื่องคอมพิวเตอร์เพียงพอกับงานไหม
แนวตอบที่ดี
  • มีการจัดซื้อ Software ถูกลิขสิทธิ์ตามระเบียบ
  • มีแผนจัดหา / ทดแทนอุปกรณ์ตามรอบที่กำหนด

คำถามลึก (บางครั้งเจอ ถ้าโฟกัส IT จริง)

  • มี DR Site (ศูนย์สำรอง) ไหม
  • Backup เก็บไว้ที่ไหน ออฟไซต์หรือเปล่า
  • เคยโดนไวรัส / แฮกไหม จัดการอย่างไร
แนวตอบปลอดภัย
  • มีการ Backup และแยกเก็บข้อมูลเพื่อลดความเสี่ยง
  • มีมาตรการป้องกัน เช่น Firewall และ Antivirus
  • มีการเฝ้าระวังเหตุการณ์ด้านความปลอดภัยอย่างสม่ำเสมอ

สูตรตอบมาตรฐาน

มี + ทำ + ตรวจสอบ + พัฒนา
ตัวอย่าง: "มีระบบสำรองข้อมูล ทำทุกวัน ตรวจสอบสม่ำเสมอ และมีการพัฒนาอย่างต่อเนื่อง"

ห้ามตอบแบบนี้

  • ไม่รู้
  • ไม่ได้ทำ
  • ยังไม่มี

เปลี่ยนเป็นแบบนี้

  • อยู่ระหว่างดำเนินการ
  • กำลังพัฒนา
  • มีแผนปรับปรุง
